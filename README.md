# DATA 221 – Assignment 3

This repository contains my solutions for **DATA 221: Introduction to Data Science – Assignment 3**.

# Contents  
The repository includes Python scripts (or a Jupyter Notebook) for each question, along with required datasets:

- `question1_assignment3_data221.py` – Statistical analysis of `ViolentCrimesPerPop` (mean, median, standard deviation, min, max, distribution comments)  
- `question2_assignment3_data221.py` – Histogram and box plot for crime data with written interpretation  
- `question3_assignment3_data221.py` – Train/test split of kidney disease dataset (70/30 split with explanations)  
- `question4_assignment3_data221.py` – KNN classifier (k = 5) with confusion matrix and evaluation metrics  
- `question5_assignment3_data221.py` – KNN comparison for k = 1, 3, 5, 7, 9 with accuracy table and overfitting/underfitting discussion  

# Data Files
- `crime.csv`  
- `kidney_disease.csv`  

# Output Files
- Generated plots (histogram and box plot)  
- Confusion matrix output  
- Accuracy comparison table for different k values  

# Notes
All code was written in Python using `pandas`, `numpy`, `matplotlib`, and `scikit-learn` as required.  
Each script corresponds directly to a question in the assignment and includes written explanations in comments where requested.  
The code runs top-to-bottom without errors when the datasets are placed in the same directory.
